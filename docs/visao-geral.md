# Visão Geral — Sistema BOOKSHARE

## Descrição

O BOOKSHARE é um sistema web/mobile desenvolvido para permitir o compartilhamento de livros físicos entre usuários, por meio de um mecanismo controlado de empréstimos e avaliações.

## Problema

Livros possuem alto custo e muitas vezes ficam ociosos após a leitura. O sistema busca aumentar a circulação desses exemplares.

## Solução

Criar uma plataforma onde usuários possam:

* cadastrar livros
* solicitar empréstimos
* aprovar solicitações
* registrar devoluções
* avaliar outros usuários

## Público-Alvo

* Estudantes
* Leitores
* Comunidades acadêmicas
* Clubes de leitura

## Objetivos

* Aumentar circulação de livros
* Reduzir custo de aquisição
* Criar rede de compartilhamento confiável
* Controlar empréstimos entre usuários

## Escopo do Sistema

Inclui:

* Cadastro de usuários
* Cadastro de livros
* Empréstimos
* Devoluções
* Avaliações
* Reputação

Não inclui:

* Venda de livros
* Pagamentos
* Entrega logística

## Atores

Usuário
Sistema

## Domínio

O domínio central é o gerenciamento de empréstimos entre usuários.

## Entidades Principais

* Usuário
* Livro
* Empréstimo
* Avaliação
* Notificação

## Diferenciais

* Sistema de reputação
* Limite dinâmico de empréstimos
* Avaliação pós-empréstimo
* Controle de disponibilidade

## Resultado Esperado

Uma plataforma colaborativa que incentive o compartilhamento de livros de forma segura e organizada.
